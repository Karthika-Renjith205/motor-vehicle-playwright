# Check Motor Vehicle Stamp Duty Test Automation

Automated UI tests for [MotorVehicleStampDuty](https://www.service.nsw.gov.au/transaction/check-motor-vehicle-stamp-duty) using **Playwright + Cucumber**.
-----------------------------

## **Setup**

1. Clone repo:

git clone 
cd MotorVehicleRegistration

2. Install dependencies:

npm init playwright@latest
npm install -D @cucumber/cucumber playwright ts-node typescript

3. Run tests : npm run cucumber
4. Run report : npm run cucumber
